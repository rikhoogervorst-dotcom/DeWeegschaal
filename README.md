# DeWeegschaal
Leg de logs van je eigen guild én die van anderen op de weegschaal. Vergelijk raid- en boss-tijden (met datum) en gemiddelde parses, bekijk per speler de parses van elke raid naast elkaar — voor healers de healing-parses — plus deaths en concrete verbeterpunten per speler en per boss, en sluit af met een ludiek taveerneverslag voor in de Discord.
